Become aFront-End Web Developer

Learn to build modern websites

How this course can help you

Whether you are just getting started as a developer or want to take your development skills to the next level, this course will expose you and teach you the hosttest technologies essential to today's front-end developers. Best practices and web standards are at the core of every topic.

If you are interested in learning Responsive Design, HTML, CSS, Javascript and other essential skills, this is the course for you.

Learn the skills that matter

HTML and CSS

There are many essential tools for building websites, but probably none more critical than HTML and CSS. Knowing how to write proper markup that is both accessible and search engine optimized, as well as styles that turn a mediocre website into a eye-catching one, are skills highly sought after.

Responsive Design

Building websites that look and work great on any device, including mobile devices, is a skill that will put you in advantage in today's market.

There are many pieces to building responsive websites and this course will walk you through each of them so you can become proficient at it.

Javascript Basics

Adding animation and interactive functionality to a website is usually achieved with Javascript. Javascript has become one of the most widely used coding languages in recent years because is powerful, lightweight and it's supported by all browsers and environments. You will be on your way to becoming proficient in Javascript.

Front-End Workflows

Having a workflow that saves you time and makes you more productive is critical to being a great web developer. Given the number of tools, processes and tasks you need to manage while development websites, automation becomes essential. You will learn how to automate your workflow so you can focus on coding, ranther than managing tools.

What makes this workshop different from the rest?

This workshop's main focus is to make you the best front-end developer you can be. Other workshops teach you a lot but without a clear goal. In today's web development environment there is clear separation between designers, front-end and back-end developers. It's typical for agencies or companies to hire developers of specific discipline.

Although you can be a full stack developer, it is extremely difficult to stay on top of the lastest trends for more than a one role at a time.
